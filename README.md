# TextArtImages
A little script which takes an image and makes a 5 "color" unicode representation of it.

## Usage
To use, run the python script from the command line like so:

```
python TextArtImages.py Path/To/Image.jpg [Output Path] [Output Width]
```

Example sytax:

```
python .\TextArtImages.py "D:\Downloads\drake.jpg" "D:\Downloads\drake.txt" 60
```
